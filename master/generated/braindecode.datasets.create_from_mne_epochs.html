braindecode.datasets.create_from_mne_epochs(list_of_epochs, window_size_samples, window_stride_samples, drop_last_window)¶

Create WindowsDatasets from mne.Epochs

Parameters
list_of_epochs: array-like

list of mne.Epochs

window_size_samples: int

window size

window_stride_samples: int

stride between windows

drop_last_window: bool

whether or not have a last overlapping window, when windows do not equally divide the continuous signal

Returns
windows_datasets: BaseConcatDataset

X and y transformed to a dataset format that is compativle with skorch and braindecode

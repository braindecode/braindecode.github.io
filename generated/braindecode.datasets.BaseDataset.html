braindecode.datasets.BaseDataset¶

class braindecode.datasets.BaseDataset(raw, description, target_name=None)¶

A base dataset holds a mne.Raw, and a pandas.DataFrame with additional description, such as subject_id, session_id, run_id, or age or gender of subjects.

Parameters
raw: mne.io.Raw
description: pandas.Series

holds additional description about the continuous signal / subject

target_name: str | None

name of the index in description that should be use to provide the target (e.g., to be used in a prediction task later on).
